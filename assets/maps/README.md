# Map PoC data

`india-ne-110m.geojson` is the India feature extracted from Natural Earth's
1:110m Admin 0 Countries dataset. Natural Earth data is public domain; this
copy is retained as a small, deterministic fixture for the experimental Map
kit rather than downloaded while rendering.

Country boundaries are cartographic data, not legal determinations. This PoC
uses the viewpoint encoded by the source dataset and must not be presented as
an authoritative boundary reference.

Source: https://github.com/nvkelso/natural-earth-vector/blob/master/geojson/ne_110m_admin_0_countries.geojson
